First install required npm packages

1. run npm install

To start application dev mode

-   1. npm run watch
-   2. npm run dev

To start application prod mode

-   1. npm run build
-   2. npm run start

To see documation of openApi, run server and go to
http://localhost:4000/static/docs.html
